# CustomiseMe UK

A Django e-commerce platform for custom-printed and embroidered clothing and accessories. Customers can browse products, submit design briefs, send in their own garments for customisation, and check out via Stripe. Orders and fulfilment are managed through a custom admin dashboard.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Django 6 (Python) |
| Database | PostgreSQL (production) / SQLite (development) |
| File Storage | Firebase Cloud Storage |
| Payments | Stripe Checkout |
| Transactional Email | Brevo (formerly Sendinblue) |
| Static Files | WhiteNoise |
| Error Tracking | Sentry |
| Server | Gunicorn |

---

## Project Structure

```
customiseproject/       # Django project root (settings, urls, wsgi)
accounts/               # Custom user model, auth views, email service
customiseapp/           # Products, carousel, design submissions, send-item requests, customisations
orderapp/               # Order + OrderItem models, Stripe webhook handling, order tracking
templates/              # HTML templates
staticfiles/            # Collected static files (generated by collectstatic)
```

## Local Development Setup

### 1. Clone and install dependencies

```bash
git clone <repo-url>
cd customiseproject
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Configure environment variables

Copy the example env file and fill in your values:

```bash
cp .env.example .env
```

See `.env.example` for all required variables and descriptions.

### 3. Run migrations

```bash
python manage.py migrate
```

### 4. Create a superuser

```bash
python manage.py createsuperuser
```


### 5. Start the development server

```bash
python manage.py runserver
```

The app will be available at `http://localhost:8000`.

## Brevo (Email) Setup

Transactional emails (order confirmations, password resets, email verification) are sent via Brevo.

1. Create a Brevo account and generate an API key under **SMTP & API → API Keys**.
2. Verify your sending domain and set `DEFAULT_FROM_EMAIL` to a verified address.

## Key Environment Variables

| Variable | Description |
|---|---|
| `SECRET_KEY` | Django secret key |
| `DEBUG` | `True` for development, `False` for production |
| `DJANGO_ENV` | Set to `production` to enable PostgreSQL |
| `DB_*` | PostgreSQL connection details |
| `FIREBASE_*` | Firebase service account credentials |
| `STRIPE_SECRET_KEY` | Stripe secret API key |
| `STRIPE_PUBLISHABLE_KEY` | Stripe publishable key (used in frontend) |
| `STRIPE_WEBHOOK_SECRET` | Stripe webhook signing secret |
| `BREVO_API_KEY` | Brevo transactional email API key |
| `SITE_URL` | Full base URL of the site (used in emails) |

See `.env.example` for the complete list.
